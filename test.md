# Content

## 性能测试
- 测试分类:
  - 性能测试: 动力
  - 负载测试: 载重
  - 压力测试: 强度
- 指标:
  - 应用系统指标(LoadRunner, SoapUI, JMeter)
    - 并发数
    - 响应时间
    - 每秒初始事物数目(TPS吞吐量) TPS = 并发数 / 响应时间
    - 每秒返回字节数(Throughout吞吐率)
    - 每秒点击次数
    - 每秒成功点击次数
    - 每秒失败点击次数
  - JVM指标(JConsole, VisualVM, JProbe, JProfiler)
  - OS指标(nmon, vmstat)
    - CPU, MEM, IO, NET
